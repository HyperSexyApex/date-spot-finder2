# MY FIRST DATE SPOT FINDER 〜掛川市版〜

**掛川市のデートスポットがすぐ決まる！**  

デートの毎回の悩み『どこ行く？』という悩みを減らせるwebアプリです。


---

## アプリ概要

| 項目 | 内容 |
|------|------|
| 種別 | ブラウザで動作するシングルページWebアプリ |
| 技術 | HTML / CSS / JavaScript |
| フレームワーク | 不使用（DOM操作・イベント委譲・localStorage管理など自前実装） |
| 対象エリア | 静岡県掛川市 |

---

##  コンセプト / 目的

デートで一番困るのは  
**「今日どこ行く？」が決まらない時間**

- ざっくりしたキーワードで候補表示
- 迷ったらランダムで決定
- 気に入った場所はお気に入り登録
- そのまま **Google Map でルート確認**

**決める→移動→楽しむ**の流れを止めないUXを目指して設計しました。

---

##  想定ユーザー

- 掛川市・周辺に住むカップル
- 掛川市の穴場スポットを見つけたい方
- 20〜30代で車/電車移動が多い層
- デートがマンネリ化してきた人
- その場で次のスポットを決めたい人


---

## 実装機能一覧

| 機能名 | 内容 |
|-------|------|
|  キーワード & カテゴリ検索 | 「公園」「カフェ」などの単語で検索可能 |
| ⭐ お気に入り登録 | Set + localStorage で永続保存 |
| 🎲 ランダム表示 | 迷ったとき1クリックで決定 |
| 📍 Google Map 連携 | 緯度経度があればピンポイント、ない場合は名称検索でフォールバック |
|  モーダル詳細表示 | カードクリックで詳細＆統一画像レイアウト表示、下線などではっきりとしたレイアウト|

---

##  技術的ポイント

- **Set** を用いて O(1) でお気に入りの管理  
  → 参照高速化＆UIと状態の分離を実現
- **localStorage** による永続化  
  → アプリを閉じてもお気に入り保持
- **イベント委譲** (`document.addEventListener`)  
  → 動的に生成される要素にも対応
- **Google Map API 連携**  
  → 緯度経度がない場合でも名称検索にフォールバック処理
- **UI/UX改善**  
  - モーダル導入
  - 画像比率の統一 (`object-fit: cover`)
  - 改行制御 (`white-space: pre-line`)

  - (JSの理解はまだあまりないためメモをすることによって理解を深めるように心がけた)

## 🚀 使い方

1. キーワード or カテゴリをクリック
2. 気になるスポットのカードを選択
3. 詳細を確認して「📍 地図で見る」を押す
4. そのままデートへGO！
